# -youtube-upload-automation
Python script that automates YouTube video publishing using PyAutoGUI and AppleScript. It auto-fills descriptions, navigates YouTube Studio, and publishes videos in bulk, reducing repetitive manual work for content creators.

This project is a Python-based automation tool that streamlines the process of publishing videos on YouTube. It uses GUI automation to automatically insert video descriptions, navigate through the upload interface, and publish content efficiently.

Built with libraries like PyAutoGUI and native macOS scripting via AppleScript, this script mimics user interactions such as mouse movements, clicks, keyboard input, and tab switching.

Key Features

* Automated description insertion for multiple videos
* Simulates full YouTube Studio publishing workflow
* Batch processing support
* macOS clipboard integration using subprocess
* Reduces repetitive manual work

Use Case

Ideal for content creators who upload gameplay walkthroughs, tutorials, or bulk video content and want to save time on repetitive publishing tasks.

Requirements

* Python 3.x
* macOS (for AppleScript support)
* Installed dependencies:
    * pyautogui

Disclaimer

This tool relies on screen coordinates and UI interactions, which may vary depending on screen resolution and YouTube interface updates. Adjust coordinates accordingly for your setup.
